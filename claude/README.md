# 🌙 Сура Аль-Мульк — Система запоминания

Одностраничный статический сайт для запоминания Суры Аль-Мульк (Коран, Сура 67) методом интервального повторения и активного вспоминания.

## ✨ Возможности

- 10 блоков по 3 аята (по намазам и ракаатам)
- 3 режима: Арабский + перевод / Только арабский / Режим запоминания
- Прогресс-трекер с localStorage (Прочитал / Выучил / Повторил)
- Блок дня (автоматически по дате)
- Поиск по намазу или номеру аята
- Мнемонические истории, крючки, мини-тесты активного вспоминания
- График интервального повторения (1/1/3/7/14/30 дней)
- Стили для печати
- Без зависимостей, без backend — только `index.html`

## 🚀 Развертывание на GitHub Pages

1. Создайте новый репозиторий на GitHub (например, `al-mulk-hifz`)
2. Загрузите `index.html` в корень репозитория
3. Перейдите в **Settings → Pages**
4. В поле **Source** выберите ветку `main`, папку `/ (root)`
5. Нажмите **Save** — сайт будет доступен по адресу:
   `https://<ваш-логин>.github.io/al-mulk-hifz/`

### Через Git (рекомендуется)

```bash
git init
git add index.html README.md
git commit -m "Initial commit: Al-Mulk memorization site"
git branch -M main
git remote add origin https://github.com/<логин>/<репозиторий>.git
git push -u origin main
```

## 💻 Открытие локально

Просто откройте `index.html` в браузере — двойной клик или:

```bash
# Python 3
python -m http.server 8000
# Затем откройте http://localhost:8000
```

> LocalStorage работает при открытии через `file://` в большинстве браузеров.

## ✏️ Как редактировать тексты аятов и переводы

Все данные находятся в JavaScript-массиве `BLOCKS` внутри `<script>` тега в конце файла.

### Изменить арабский текст аята

Найдите нужный блок по номеру (`id: 1` и т.д.), затем в `arabicText`:

```javascript
{ n: 1, ar: "تَبَارَكَ الَّذِي...", ru: "Перевод на русский..." }
```

Замените значение поля `ar` на нужный текст.

### Изменить русский перевод

Переводы основаны на материалах [umma.ru](https://umma.ru). Отредактируйте поле `ru` в соответствующем объекте.

### Добавить или изменить мнемонику / историю

Для каждого блока доступны поля:
- `story` — мнемоническая история
- `hook` — крючок-смысл
- `howToRemember` — советы по запоминанию
- `recallQuestions` — массив из 3 вопросов для мини-теста
- `keyMeaning` — ключевой смысл блока
- `emojis` — 3 эмодзи-якоря
